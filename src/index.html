HOW TO USE

GET /compare?name1=[NAME OF PLAYER1]&name2=[NAME OF PLAYER2]&year=[YEAR TO COMPARE]&stats=[ONE OR MORE OF THE OPTIONS]

THE OPTIONS ARE

  • age = Player age
  • team_id = Team name abbreviation
  • pos = Player position
  • g = Games played in the season
  • gs = Games player have started
  • mp_per_g = Minutes per game
  • fg_per_g = Field goals per game
  • fga_per_g = Field goals attempts per game
  • fg_pct = Field goals percentage
  • fg3_per_g = 3-Point Field goals per game
  • fg3a_per_g = 3-Point Field goals attempts per game
  • fg3_pct = 3-Point Field goals percentage
  • fg2_per_g = 2-Point Field goals
  • fg2a_per_g = 2-Point Field goals attempts per game
  • fg2_pct = 2-Point Field goals percentage
  • efg_pct = Effective field goal percentage
  • ft_per_g = Free throws per game
  • fta_per_g = Free throws attempts per game
  • ft_pct = Free throws percentage
  • orb_per_g = Offensive rebounds per game
  • drb_per_g = Deffensive rebounds per game
  • trb_per_g = Total rebounds per game
  • ast_per_g = Assists per game
  • stl_per_g = Steal per game
  • blk_per_g = Blocks per game
  • tov_per_g = Turnovers per game
  • pf_per_g = Personal fouls per game
  • pts_per_g = Points per game
